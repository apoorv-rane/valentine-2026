# Valentine's Day Proposal Website

An interactive, animated Valentine's Day proposal website built with vanilla HTML5, CSS3, and JavaScript.

## Features

- **Animated Loader** - 5-second pulsing heart animation with anticipation-building text
- **Typed Greeting** - Character-by-character typing effect for personalized message
- **Countdown Timer** - Real-time countdown to Valentine's Day with timezone support
- **Progressive Reveal** - Scroll-triggered animations that reveal content as you scroll
- **Interactive Buttons** - Playful Yes/No interaction with escalating responses
- **Confetti Animation** - Celebratory confetti burst with 80+ colorful pieces
- **Background Music** - Autoplay-friendly music toggle with visual feedback
- **Floating Hearts** - Ambient floating heart animations throughout the page
- **Mobile Responsive** - Fully optimized for mobile and desktop experiences
- **Scroll Hint** - Mobile-only indicator showing scrollable content

## Technical Stack

- **HTML5** - Semantic structure
- **CSS3** - Animations, transitions, and responsive design
- **Vanilla JavaScript** - Interactive logic without dependencies
- **Google Fonts** - Poppins, Playfair Display, Cormorant Garamond

## Customization

- **Text Content**: Edit the greeting and messages in `index.html`
- **Colors**: Modify the color scheme in `styles.css` (primary: `#e63946`)
- **Timing**: Adjust animation durations in `styles.css` and loader delay in `script.js`
- **Images**: Replace `images/us1.jpg` and `images/us2.jpg` with your own
- **Music**: Swap `music/song.m4a` with your preferred track
